Estacione aqui

Esse projeto foi desenvolvido no 4° semestre da faculdade, sua função é gerenciar a entrada e saida de veiculos de um estacionamento, o site foi desenvolvido em PHP seguindo o padrão MVC

Aerocris

O website da empresa Aerocris foi desenvolvido em um Hackathoon na Fatec de JAHU . Nele os participantes deveriam entregar um site em Wordpress em dois dias. A ideia deste site é passar aos usuários uma ideia de segurança e qualidade dos produtos, além de permitir uma aproximação melhor com a empresa.

My spot

Projeto desenvolvido no 5° semestre, este projeto tem a finalidade de armezar seus lugares favoritos, atraves de um campo de busca os usuarios podem pesquisar endereços e adiciona-los para uma consulta no futuro. O projeto foi desenvolvido em PHP seguindo o padrão MVC.

Browser Name

Browser Name foi codificado em Javascript, tem como função identificar o nome e versão do navegador. Tambem é possivel adicionar uma classe css para especificar como um elemento deve se comportar em diversos navegadores.
